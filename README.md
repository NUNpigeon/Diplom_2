## Описание проекта


Проект содержит автоматизированные тесты для API сервиса **Stellar Burgers**. Проверяются ключевые эндпоинты:
- создание пользователя;
- авторизация;
- создание заказов.


## Функционал, покрытый тестами


### 1. Создание пользователя
- создание уникального пользователя;
- попытка регистрации уже существующего пользователя;
- регистрация с пропущенным обязательным полем (`email`, `пароль`, `имя`).


### 2. Авторизация пользователя
- успешный вход с корректными данными;
- ошибка при входе с неверными логином/паролем.


### 3. Создание заказа
- создание заказа с авторизацией;
- создание заказа без авторизации;
- заказ с валидными ингредиентами;
- заказ без ингредиентов;
- заказ с некорректным хешем ингредиентов.


## Технические детали


**Используемые технологии:**
- Java 11+;
- Maven 3.8+;
- JUnit 4.13;
- REST Assured 5.3+;
- Allure Reports 2.22+.
